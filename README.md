O projeto Controle de Estoque consiste no desenvolvimento de um sistema voltado para o gerenciamento de materiais utilizados em ambientes acadêmicos da área da saúde, especialmente em aulas práticas de medicina. A aplicação permite registrar, organizar e acompanhar a quantidade de insumos disponíveis, como seringas, gazes, luvas e outros materiais utilizados nos laboratórios, facilitando o controle e evitando desperdícios ou falta de recursos.

A proposta do sistema é oferecer uma solução simples e eficiente para o controle de entrada e saída de materiais, permitindo a visualização do estoque atual, o cadastro de novos itens e o registro das movimentações realizadas. Dessa forma, o sistema contribui para uma melhor organização dos insumos utilizados nas atividades práticas, auxiliando alunos, professores e responsáveis pelos laboratórios no gerenciamento dos materiais.
